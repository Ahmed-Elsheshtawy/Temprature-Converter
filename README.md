# Temprature Converter

Steps to Use App:

1. Click "Download raw" on the "Temprature Converter.apk" file.
2. On your Android device, get the downloaded file.
3. Install the file on your Android device.
4. If a warning pops up such as "This file might contain harmful content" then click "Install anyway" (the file is safe).
5. Have fun converting the temperature!
